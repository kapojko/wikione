WikiOne
-------

The simplest wiki+tracking system written on PHP+MySql.

Author Kapoyko Yu. A., Russia.

This work is published into the Public Domain.

INSTALLATION:

1. Unzip, checkout or export all to 'www' folder, ex. 'www/tracker';
2. Copy 'config-dist.php' to 'config.php' and set all variables to approriate values;
3. Open '../install.php' file with browser;
4. DONE.

UNINSTALLATION:
1. Open '../uninstall.php' file with browser;
2. DONE.
